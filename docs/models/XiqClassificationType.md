# extremecloudiq.model.xiq_classification_type.XiqClassificationType

The classification type for VLAN assignment. LOCATION: Classify based on device location (e.g., floor, building). CLOUD_CONFIG_GROUP: Classify based on cloud configuration groups. IP_ADDRESS: Classify based on specific IP addresses. IP_SUBNET: Classify based on IP subnet ranges (e.g., 192.168.1.0/24). IP_RANGE: Classify based on IP address ranges (e.g., 192.168.1.1-192.168.1.100).
